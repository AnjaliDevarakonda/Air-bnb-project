# London Airbnb Market Analysis

## Overview
End-to-end data science project analyzing Airbnb listings in London.

## Project Structure
- `data/` - Dataset (listings.csv)
- `notebooks/` - EDA and model training
- `app/` - Streamlit dashboard

## Key Features
- Interactive map of listings
- Price analysis by neighborhood
- Host performance insights
- Price prediction model

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `streamlit run app/app.py`
